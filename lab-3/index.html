ITC505 Lab 3 - Serving Custom Fonts

@font-face Rule

This rule in CSS allows us to define a new font family, such as 'ProtestRevolution-Regular'. It specifies where the font files are located (woff2, woff, ttf formats) using URLs. These different formats ensure the font works well on various browsers and systems.

Font Family in Body Style

We set the body of our webpage to use 'ProtestRevolution-Regular' as the main font. If this custom font isn't available on a user's device, the browser will automatically switch to using standard fonts like Arial, Helvetica, or sans-serif.

Deployment

Place the font files in a fonts folder on your server. Upload your HTML file and font files to your web server. Check that your webpage shows the custom font correctly when accessed. This method ensures your custom font is delivered reliably to everyone visiting your webpage.
